# Python Scripts

## Run Scripts

1. Activate the `venv`:

   ```shell
   source venv/bin/activate
   ```

2. Run `main.py`:

   ```shell
   python3 main.py
   ```

3. Deactivate the `venv`

   ```shell
   deactivate
   ```
