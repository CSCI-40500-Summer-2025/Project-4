# Python Scripts

## Run Scripts

Must have `python3` installed.

1. Create the virtual environment.

   ```shell
   python3 -m venv venv
   ```

2. Activate the `venv`:

   ```shell
   source venv/bin/activate
   ```

3. Install dependencies.

   ```shell
   pip install -r requirements.txt
   ```

4. Run `main.py`:

   ```shell
   python3 main.py
   ```

5. Deactivate the `venv`

   ```shell
   deactivate
   ```
